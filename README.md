# Portfolio

Personal website where I showcase my creative and technical work. Built with Next.js, React Three Fiber, and GSAP. The site features an interactive 3D hero section and smooth scroll-driven animations.

## 💻 Live Demo

<a href="https://kubawalczak.vercel.app" target="_blank">https://kubawalczak.vercel.app</a>

## ✨ Features

- Interactive 3D hero section (React Three Fiber)
- Smooth animations (GSAP)
- Responsive design
- Optimized performance with Next.js
- Optimized for both desktop and mobile experiences

## 🛠 Tech Stack

- Next.js
- React
- React Three Fiber
- GSAP
- Tailwind CSS
